# Run Test Scripts in Postman

## Description
### ** Run the Collection**
1. Go to the **Collection Runner** in Postman.
2. Select the imported collection.
3. Adjust the request parameters or iterations as needed.
4. Click **Run** to execute all test requests in sequence.
